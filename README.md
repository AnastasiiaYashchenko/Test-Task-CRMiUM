1. Обліковий запис Zoho CRM
2. Модуль Deals
3. Поле типу Decimal з назвою "Курс валют" та API name - "deal_rate" у модулі Deals
4. Проєкт повинен містити такі файли:
index.html
script.js
style.css
ua.json
en.json
README.md
5. При створенні віджету виберіть hosting - external, base URL - http://127.0.0.1:5500/
6. Запустіть проект за допомогою консолі в Visual Studio Code  та скрипта start index.html
7. Відкрийте Zoho crm  та раніше створенний віджет
